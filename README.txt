ВАЖНО:
PWA работает ТОЛЬКО по https:// или на localhost.
То есть надо разместить на хостинге (GitHub Pages / Netlify / Vercel) или запускать локально на Mac через http.server.

iPhone установка:
1) Открой в Safari по URL
2) Поделиться → На экран «Домой»
